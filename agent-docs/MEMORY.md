# Memory

## Overview

This file documents how memory works inside the AI system.

Memory is one of the most important parts of an agent architecture because it affects:
- personalization,
- context retention,
- planning quality,
- and long-term reliability.

This document should explain:
- what gets stored,
- how retrieval works,
- and how memory is managed safely.

---

# Memory Types

## Working Memory

Short-term context used during active tasks.

### Characteristics

- Temporary
- Fast access
- Limited size

---

## Episodic Memory

Stores prior interactions and events.

### Examples

- Previous conversations
- Completed tasks
- User preferences

---

## Semantic Memory

Stores extracted knowledge and facts.

### Examples

- Stable project information
- Learned relationships
- System rules
