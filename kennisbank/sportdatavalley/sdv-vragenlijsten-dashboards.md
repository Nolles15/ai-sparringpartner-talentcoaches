# Vragenlijsten dashboards

Voor: coaches, sporters en onderzoekers. Onderdeel van SportData Valley: analyseren.

Je kunt de data van je vragenlijsten analyseren via de **Vragenlijsten**-dashboards op de **Analyse**-pagina.

Het dashboard biedt navigatie naar de verschillende vragenlijsten (trainingsdagboek, dagelijkse vragenlijst, SQUASH-vragenlijst). De gedetailleerde stappen per dashboard staan op de gerelateerde pagina's in de categorie Vragenlijsten (zie "Dashboards voor data van vragenlijsten").

Bron: https://tutorial.sportdatavalley.nl/analyse/vragenlijsten-dashboards/
