# SDV API (API-token aanmaken)

Voor: ontwikkelaars en onderzoekers. Onderdeel van SportData Valley: voor ontwikkelaars.

Met de SDV-API kun je persoonlijke integraties maken met andere platforms, of data waar jij toegang toe hebt binnen SportData Valley naar andere platforms halen.

Zo maak je een API-token aan:
1. Ga naar je profielpagina.
2. Selecteer het tabblad **Geavanceerd**.
3. Klik op **API-token toevoegen**.
4. Geef het token een naam en klik op **API-token opslaan**.

Alle aangemaakte tokens verschijnen daarna op dezelfde pagina ter referentie.

Gedetailleerde API-documentatie staat op https://app.sportdatavalley.nl/developer

Bron: https://tutorial.sportdatavalley.nl/analyse/voor-ontwikkelaars/
