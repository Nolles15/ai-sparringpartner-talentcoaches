# Een kopie (snapshot) van je dataset(s) maken

Voor: coaches, onderzoekers en sporters. Onderdeel van SportData Valley: data-beheer.

Een snapshot is een gecomprimeerde map met een kopie van de datasets die je hebt geselecteerd.

**Een snapshot maken:**
1. Ga naar de **Data**-pagina.
2. Vink alle datasets aan die je in de snapshot wilt opnemen.
3. Geef de snapshot een naam in de snapshot-sectie die boven de datasetlijst verschijnt.
4. Klik op **Creëer snapshot**.

**Snapshot downloaden:**
- Ga naar de **Delen**-pagina.
- Scroll omlaag en klik op de knop **Downloaden** bij je snapshot.

**Snapshot delen:**
- Klik op de **Delen**-pagina op **Delen** en kies de gebruikers of groepen waarmee je de snapshot wilt delen.
- Let op: een gedeelde snapshot kun je niet meer ongedaan maken.
