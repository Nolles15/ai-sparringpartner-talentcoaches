# De JupyterHub-terminal gebruiken

Voor: onderzoekers/Premium-research accounts. Onderdeel van SportData Valley: analyseren (JupyterHub).

De terminal van JupyterHub werkt op dezelfde manier als de command line of shell van je besturingssysteem (Windows, macOS, Linux).

Je gebruikt de terminal voor:
1. Packages installeren.
2. Een virtuele omgeving maken (zie "Een virtuele omgeving maken in JupyterHub").
3. GitHub of GitLab gebruiken in je JupyterHub.

Bron: https://tutorial.sportdatavalley.nl/jupyter-hub/de-jupyter-hub-terminal-gebruiken/
