# Een nieuw Jupyter notebook aanmaken

Voor: onderzoekers/Premium-accounts. Onderdeel van SportData Valley: analyseren (JupyterHub).

Zo maak je een nieuw notebook aan:

1. Klik op een van de drie soorten notebooks in het startscherm, of op het **+**-teken aan de linkerkant van je scherm. Via het startscherm wordt automatisch een notebook geopend.
2. Kies een programmeertaal: **Python**, **R** of **Octave**. Kies de taal waarmee jij het beste overweg kunt.

Bron: https://tutorial.sportdatavalley.nl/jupyter-hub/een-nieuwe-jupyter-notebook/
